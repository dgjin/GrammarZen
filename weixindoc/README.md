# 微信公众号文章素材说明

## 文件说明

- **grammarzen-wechat-article.md**：公众号正文（Markdown），可直接复制到公众号后台或先转为富文本。
- **images/**：配图目录。

## 配图清单

| 文件名 | 说明 | 备注 |
|--------|------|------|
| `grammarzen-modes.png` | 六种校对模式示意图 | 已生成，可直接用 |
| `grammarzen-workflow.png` | 使用流程三步示意 | 已生成，可直接用 |

| `file-upload.png` | 文件上传界面 | 同上 |

## 发布前建议

1. **替换三张截图**：用实际产品的主界面、上传界面、结果页截图替换 `main-interface.png`、`file-upload.png`、`issue-list.png`，文章内对应说明可删掉“建议替换为…”的提示。
2. **动图**：按文章第七节建议录制 1～2 段操作流程 GIF，插入到「主界面一览」或「使用流程」附近，阅读体验更好。
3. **上传到公众号**：在公众号后台「图文消息」里，把 `images/` 中的图片逐一上传到「图片」库，再在正文中插入对应图片即可。
